# 3D Model Validation Checklist

## Unisex Merino T-shirt Model

### Visual Quality
- [ ] Model accurately represents product design
- [ ] Fabric draping looks realistic
- [ ] Proportions match actual product
- [ ] Details (collar, sleeves, hem) are well-defined

### Textures
- [ ] Merino wool texture is realistic
- [ ] Normal maps create appropriate surface detail
- [ ] Color variants (Forest Green, Beige, Black) display correctly
- [ ] Texture resolution is sufficient without being excessive

### Performance
- [ ] File size is under 2MB
- [ ] Polygon count is optimized (<10,000)
- [ ] Loading time is under 2 seconds
- [ ] Frame rate remains above 30 FPS on mobile

### Technical Requirements
- [ ] UV mapping is properly implemented
- [ ] Model is centered at origin
- [ ] Scale is appropriate
- [ ] Animations (if any) are smooth

### Integration
- [ ] Model loads correctly in 3D viewer
- [ ] Color switching works properly
- [ ] Rotation and zoom function as expected
- [ ] Mobile touch controls work correctly
